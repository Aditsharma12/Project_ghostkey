Field Operative Guide

Mission Parameters

This guide outlines the operational procedures for the Ghostkey suite. Each tool provides a layer of security through encryption or concealment. Correct procedure is critical for data integrity.

Protocol: Text Encryption & Decryption

  1. Select Text Encrypt or Text Decrypt from the tool dial.
  2. Input the text into the designated data field.
  3. Provide a secure, memorable keyword. This keyword is your only key.
  4. Execute the task. The output will be the encrypted/decrypted text.

Protocol: Image Steganography

  1. To Encrypt: Select Image Encrypt. Choose an image file, type the secret message to hide, and provide a keyword. The output will be a new image containing the hidden message.
  2. To Decrypt: Select Image Decrypt. Upload the image that contains the hidden data and provide the exact same keyword used to encrypt it.

Protocol: Audio Steganography (Advanced)

  1. To Encrypt: Select Audio Encrypt. Choose an image file to convert into an audio signal, and provide a keyword.
  2. You will receive two files: a .WAV audio file and a critical .NPY shape file. You must save both files.
  3. To Decrypt: Select Audio Decrypt. Upload the .WAV audio file, the corresponding .NPY shape file, and the correct keyword to recover the original image.

Critical Warning: The .NPY file is essential for decryption. Loss of this file will result in permanent data loss.
